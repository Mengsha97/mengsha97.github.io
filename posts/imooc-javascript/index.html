隐式转换

+和-

巧用+/-规则转换类型

num-0 //将字符转换成数字
num+'' //将数字转换成字符

a==b

以下都为true

"1.23" == 1.23
0 == false
null == undefined
new Object() == new Object()
[1,2] == [1,2]

a === b

先比较类型

  • 类型不同，返回false
  • 类型相同：（以下为true)
    • null === null
    • undefined === undefined
    • NaN !== NaN 对象不等，例如：
    • new Object() !== new Object()
    • [1,2] != [1,2]

类型转换和比较

  • null == undefined 相等
  • number == string 转number 1 == “1.0” //true
  • boolean == ? 转number 1 == true //true
  • object == number|string 尝试对象转为基本类型，new String(“hi”) == “hi” //true 其他false
