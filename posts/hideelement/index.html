用CSS隐藏一个元素的方法

Simona  收录于  类别 学习记录
 2020-07-19  2020-07-19  约 25 字  预计阅读 1 分钟 
警告
本文最后更新于 2020-07-19，文中内容可能已过时。
  1. CSS display 的值是 none。
  2. type=“hidden"的表单元素。
  3. 宽度和高度都显式设置为 0。
  4. 一个祖先元素是隐藏的，该元素是不会在页面上显示
  5. CSS visibility 的值是 hidden
  6. CSS opacity 的指是 0

如果元素中占据文档中一定的空间,元素被认为是可见的。 可见元素的宽度或高度，是大于零。 元素的 visibility: hidden 或 opacity: 0 被认为是可见的，因为他们仍然占用空间布局。
